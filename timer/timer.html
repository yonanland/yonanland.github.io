Javascript Timer Exercises

  1. What is the differencebetween the following 2 statements?
    • SetTimeout(booyah, 2000);
      • >>> Because we don't have the () after booyah, it calls the function after "waiting" the timer goes off after 2 seconds.

    • SetTimeout(booyah(), 2000);
      • >>> Because we have the () after booyah, it calls the function booyah immediately, rather than waiting for 2 seconds (2000ms)

  2. What do the following 2 alerts display (answer without running the code)? 
                  var myfunc = function(a, x) {
                return a * x;
              };
              var x = myfunc(2, 3);
              var y = myfunc;
              alert(x);
              alert(y(2,3));

    >>> In both cases the alerts will display 6.
